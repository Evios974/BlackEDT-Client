# BlackEDT-Client

BlackEDT-Client is the client-side application for BlackEDT, a calendar application for the IUT of St-Pierre.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [License](#license)

## Installation

To install the requirements for BlackEDT-Client, follow these steps:

1. Clone the repository to your local machine.
2. Install the required dependencies using `bun install`.
3. Start the client using `bun start`.

## Usage

You can launch the client via `bun start` or `electron .`.
